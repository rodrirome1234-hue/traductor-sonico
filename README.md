# Traductor Sónico

## Descripción

Traductor Sónico es una aplicación web desarrollada con HTML, CSS y JavaScript que ayuda a productores musicales a convertir descripciones abstractas de sonido en parámetros técnicos de diseño sonoro.

Por ejemplo:

Entrada:
"Mercurio líquido"

Salida:

* Oscillator: Metallic Wavetable
* Filter: Low Pass
* LFO: Slow Movement
* FX: Chorus + Reverb

## Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Funcionamiento

1. El usuario introduce una descripción sonora.
2. JavaScript analiza palabras clave.
3. La aplicación genera recomendaciones técnicas de síntesis y procesamiento.
4. Los resultados se muestran dinámicamente en pantalla.

## Prompt Engineering

La lógica del proyecto se inspira en el uso de prompts utilizados para traducir conceptos abstractos en parámetros técnicos de producción musical.

Ejemplo:

"Actúa como un diseñador sonoro profesional. Convierte la siguiente descripción abstracta en recomendaciones de osciladores, filtros, modulación y efectos."
