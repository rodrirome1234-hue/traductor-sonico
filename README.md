# Traductor Sónico

## Descripción

Traductor Sónico es una aplicación web desarrollada con HTML, CSS y JavaScript que ayuda a productores musicales a convertir descripciones abstractas de sonido en parámetros técnicos de diseño sonoro.

La herramienta recibe una descripción escrita por el usuario y genera recomendaciones relacionadas con síntesis, filtrado, modulación y efectos.

---

## Objetivo

Facilitar el proceso creativo dentro del diseño sonoro, transformando ideas subjetivas en parámetros técnicos que puedan utilizarse en sintetizadores y estaciones de trabajo de audio digital (DAWs).

---

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GitHub

---

## Funcionamiento

1. El usuario introduce una descripción sonora.
2. JavaScript captura el texto ingresado.
3. El sistema analiza palabras clave mediante estructuras condicionales.
4. Se generan recomendaciones técnicas de diseño sonoro.
5. Los resultados se muestran dinámicamente en pantalla.

---

## Lógica del Proyecto

El proyecto utiliza:

- Variables
- Eventos (`addEventListener`)
- Manipulación del DOM
- Condicionales (`if / else`)
- Plantillas literales (Template Strings)

Estas herramientas permiten generar diferentes respuestas dependiendo de la descripción ingresada por el usuario.

---

## Estrategia de Prompt Engineering

La lógica conceptual del proyecto se inspira en el siguiente prompt:

> Actúa como un diseñador sonoro profesional. Convierte una descripción abstracta de sonido en recomendaciones técnicas de osciladores, filtros, modulación y efectos.

Este enfoque permite traducir conceptos subjetivos a parámetros útiles para producción musical y diseño sonoro.

---

## Ejemplos de Uso

### Entrada

Mercurio líquido

### Salida

- Filter: Low Pass con resonancia moderada

- FX: Chorus + Reverb

Bajo agresivo
